SOC Alert Investigation Project – LetsDefend

This project contains investigation of multiple SOC alerts using the LetsDefend SOC simulator.

Cases Included:
1. Web Attack (CVE Exploit Investigation)
2. Phishing Email Analysis
3. Brute Force Login Attack
4. Malware / Suspicious Powershell Execution

Skills Used:
- Log Analysis
- SIEM Investigation
- Threat Intelligence (VirusTotal,MXtoolbox )
- Email Security Analysis
- Incident Response
- Endpoint Containment

This project demonstrates practical SOC analyst workflow and alert triage process.
